# Whiteout Survival Knowledge Domains

> Defines the official knowledge domains for Whiteout Survival within the ODIN platform.

---

# 🇺🇸 English (Official)

## Purpose

Knowledge is organized into independent domains.

Each domain owns its entities, relationships, validation rules, and data.

This separation improves maintainability and scalability.

---

# Current Domains

| Domain    | Description                                                    |
| --------- | -------------------------------------------------------------- |
| Heroes    | Hero profiles, skills, stats, generations, and recommendations |
| Buildings | Buildings, upgrades, costs, and requirements                   |
| Furnace   | Furnace progression and unlocks                                |
| Research  | Technologies, research trees, and prerequisites                |
| Troops    | Troop types, tiers, and statistics                             |
| Pets      | Pets, abilities, and bonuses                                   |
| Items     | Consumable and permanent items                                 |
| Resources | Food, Wood, Coal, Iron, Steel, etc.                            |
| Events    | Game events and schedules                                      |
| Alliance  | Alliance systems and features                                  |
| State     | State-level mechanics                                          |
| Combat    | Battle mechanics and formations                                |
| Economy   | Resource management and optimization                           |

---

# Domain Principles

Every domain must:

- Have a single responsibility.
- Avoid duplicated knowledge.
- Follow the ODIN Knowledge Taxonomy.
- Reference verified sources.
- Support versioning.

---

# Future Expansion

Additional domains may be introduced as the game evolves.
