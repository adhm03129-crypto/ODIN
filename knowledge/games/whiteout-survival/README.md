---

# 🇺🇸 English (Official)

## Purpose

This directory contains the official knowledge base for **Whiteout Survival**.

It serves as the canonical source of truth for all game-specific information used by the ODIN platform.

All AI reasoning, OCR interpretation, analytics, planning, and recommendation services depend on this knowledge.

---

# Objectives

The knowledge base aims to:

- Organize game knowledge into logical domains.
- Maintain verified and versioned information.
- Support explainable AI recommendations.
- Enable long-term maintainability.
- Separate game knowledge from platform logic.

---

# Governance
