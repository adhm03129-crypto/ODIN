# Standards

The **Standards** directory contains the official rules and conventions that govern the entire ODIN platform.

These standards apply to:

- Documentation
- Source Code
- AI Prompts
- JSON
- YAML
- Markdown
- File Naming
- Directory Structure
- Versioning
- Release Process

Every contributor (human or AI) must follow these standards.

# Principles

The standards are designed to ensure:

- Consistency
- Maintainability
- Automation
- Scalability
- Enterprise Quality

These standards take precedence over personal preferences.
